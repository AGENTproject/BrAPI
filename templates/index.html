BrAPI Agent - endpoints

  • /samples - retrieve a list of all of the samples, e.g. /genotyping/brapi/v2/samples
  • /samples/reference_id - retrieve a sample searched by referenceID(ex biosample id: SAMEA*), e.g. /genotyping/brapi/v2/samples/SAMEA111493800
  • /callsets - retrieve a list of all callsets, e.g. /genotyping/brapi/v2/callsets
  • /callsets/reference_id - retrieve a callset searched by referenceID, e.g. /genotyping/brapi/v2/callsets/SAMEA114474862
