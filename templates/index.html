BrAPI Agent - endpoints

  • /samples - retrieve a list of all of the samples, e.g. /genotyping/brapi/v2/samples
  • /samples/reference_id - retrieve a sample searched by referenceID(ex biosample id: SAMEA*), e.g. /genotyping/brapi/v2/samples/SAMEA111493800
  • /callsets - retrieve a list of all callsets, e.g. /genotyping/brapi/v2/callsets
  • /callsets/reference_id - retrieve a callset searched by referenceID, e.g. /genotyping/brapi/v2/callsets/SAMEA114474862
  • /attributes - retrieve a list of all attributes, e.g. /genotyping/brapi/v2/attributes
  • /attributes/reference_id - retrieve a attribute searched by referenceID(ex ATTRIBUTEDBID:), e.g. /genotyping/brapi/v2/callsets/10352
  • /attributevalues - retrieve a list of all attributevalues, e.g. /genotyping/brapi/v2/attributevalues
  • /attributevalues/reference_id - retrieve a attributevalue searched by referenceID(ex GERMPLASMDBID:), e.g. /genotyping/brapi/v2/callsets/121888
