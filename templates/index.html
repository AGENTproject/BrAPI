AGENT BrAPI Implementation

What is BrAPI?

The Breeding API (BrAPI) Project is an effort to enable interoperability among agricultural databases and tools. BrAPI is a standardized RESTful web service API specification for exchanging scientific data. This community driven standard is free to be used by anyone interested in agricultural and scientific data management.

Source Code

The source code for these dedicated endpoints is hosted in a private GitHub repository: AGENTproject/BrAPI

BrAPI Documentation

BrAPI Core

The BrAPI Core module contains high level entities used for organization and management. This includes Programs, Trials, Studies, Locations, People, and Lists

V2.1 - GitHub - SwaggerHub - Apiary

BrAPI Phenotyping

The BrAPI Phenotyping module contains entities related to phenotypic observations. This includes Observation Units, Observations, Observation Variables, Traits, Scales, Methods, and Images

V2.1 - GitHub - SwaggerHub - Apiary

BrAPI Genotyping

The BrAPI Genotyping module contains entities related to genotyping analysis. This includes Samples, Markers, Variant Sets, Variants, Call Sets, Calls, References, Reads, and Vendor Orders

V2.1 - GitHub - SwaggerHub - Apiary

BrAPI Germplasm

The BrAPI Germplasm module contains entities related to germplasm management. This includes Germplasm, Germplasm Attributes, Seed Lots, Crosses, Pedigree, and Progeny

V2.1 - GitHub - SwaggerHub - Apiary

Endpoints

BrAPI-Core

  • /serverinfo - retrieve server information, e.g. /genotyping/brapi/v2/serverinfo
  • /studies - retrieve a list of all of the studies, e.g. /genotyping/brapi/v2/studies
  • /studies/{studyDbId} - retrieve a study searched by studyDbId, e.g. /genotyping/brapi/v2/studies/57143

BrAPI-Phenotyping

  • /methods - retrieve a list of all of the methods, e.g. /genotyping/brapi/v2/methods
  • /methods/{methodDbId} - retrieve a method searched by methodDbId, e.g. /genotyping/brapi/v2/methods/11259
  • /variables - retrieve a list of all of the observation variables, e.g. /genotyping/brapi/v2/variables
  • /variables/{observationVariableDbId} - retrieve an observation variable searched by observationVariableDbId, e.g. /genotyping/brapi/v2/variables/11259
  • /observations - retrieve a list of all of the observations, e.g. /genotyping/brapi/v2/observations
  • /observations/{observationDbId} - retrieve an observation searched by observationDbId, e.g. /genotyping/brapi/v2/observations/1376649
  • /observationunits - retrieve a list of all of the observation units, e.g. /genotyping/brapi/v2/observationunits
  • /observationunits/{observationUnitDbId} - retrieve an observation unit searched by observationUnitDbId, e.g. /genotyping/brapi/v2/observationunits/1376649
  • /scales - retrieve a list of all of the scales, e.g. /genotyping/brapi/v2/scales
  • /scales/{scaleDbId} - retrieve a scale searched by scaleDbId, e.g. /genotyping/brapi/v2/scales/11259
  • /traits - retrieve a list of all of the traits, e.g. /genotyping/brapi/v2/traits
  • /traits/{traitDbId} - retrieve a trait searched by traitDbId, e.g. /genotyping/brapi/v2/traits/11981

BrAPI-Genotyping

  • /samples - retrieve a list of all of the samples, e.g. /genotyping/brapi/v2/samples
  • /samples/{sampleDbId} - retrieve a sample searched by sampleDbId, e.g. /genotyping/brapi/v2/samples/3135289
  • /callsets - retrieve a list of all of the callsets, e.g. /genotyping/brapi/v2/callsets
  • /callsets/{callSetDbId} - retrieve a callset searched by callSetDbId, e.g. /genotyping/brapi/v2/callsets/SAMEA10401783

BrAPI-Germplasm

  • /attributes - retrieve a list of all of the attributes, e.g. /genotyping/brapi/v2/attributes
  • /attributes/{attributeDbId} - retrieve an attribute searched by attributeDbId, e.g. /genotyping/brapi/v2/attributes/11259
  • /attributevalues - retrieve a list of all of the attribute values, e.g. /genotyping/brapi/v2/attributevalues
  • /attributevalues/{attributeValueDbId} - retrieve an attribute value searched by attributeValueDbId, e.g. /genotyping/brapi/v2/attributevalues/1376649
  • /germplasm - retrieve a list of all of the germplasm, e.g. /genotyping/brapi/v2/germplasm
  • /germplasm/{germplasmDbId} - retrieve a germplasm searched by germplasmDbId, e.g. /genotyping/brapi/v2/germplasm/AW_77936
